1. Описание:
Программа позволяет пользователю выбирать цвет с помощью элемента <input type="color">, а затем отображает его в разных цветовых моделях: RGB, HLS, CMYK. Пользователь также может вручную вводить значения для цветовых моделей (RGB, HLS, CMYK), и программа автоматически обновляет цвет, отображаемый на экране. Если введенные значения некорректны (выходят за допустимые пределы), выводится предупреждение, и значения устанавливаются в 0.

2. Структура программы:
HTML файл (index.html): Содержит базовую разметку страницы, поля ввода для выбора цвета, формы для ввода значений RGB, HLS, CMYK, а также элемент отображения цвета и связанные с ним значения цветовых моделей.

CSS файл (style.css): Стилизация веб-страницы (не предоставлена в коде, предполагается базовая стилизация).

JS файл (logic.js): Логика обработки изменения цвета, обновления представления и конвертации между цветовыми моделями.

Electron приложение: Программа используется внутри Electron приложения для создания настольного интерфейса.

3. Основные функции:
1) Выбор системы цвета:

* Используя выпадающий список, пользователь может выбрать одну из трех систем: RGB, HLS или CMYK.
* В зависимости от выбора, отображаются соответствующие поля для ввода значений.

2) Конвертация цвета и обновление отображения:

При выборе цвета или изменении значений в полях ввода, происходит пересчет и отображение цвета в остальных цветовых системах.

3) Конвертация между цветовыми системами:

В программе есть функции для преобразования цветов между различными цветовыми моделями:

RGB в CMYK: Преобразует значения RGB в CMYK.
RGB в HLS: Преобразует значения RGB в HLS.
CMYK в RGB: Преобразует CMYK обратно в RGB.
HLS в HEX: Преобразует HLS в HEX цвет.

4) Предупреждения:

Если пользователь вводит некорректные значения (например, RGB значения вне диапазона 0-255), программа показывает предупреждение и сбрасывает значения на 0.

5) Electron настройки:

Для запуска программы на настольном приложении используется Electron. Окно создается с фиксированными размерами и без строки меню.

4.Ключевые элементы страницы:
colorPicker: Элемент выбора цвета.
rgbInputs, hlsInputs, cmykInputs: Поля ввода для различных цветовых систем.
colorDisplay: Элемент отображения выбранного цвета.
rgbValue, hlsValue, cmykValue: Поля для отображения значений цветовых моделей.
Warning: Элемент для отображения предупреждений об ошибках.
5. Использование:
Пользователь выбирает цвет с помощью элемента выбора цвета или вводит значения вручную для одной из цветовых моделей (RGB, HLS, CMYK).
Программа отображает цвет и автоматически обновляет его значения в остальных системах.
Пользователь может изменить систему и ввести новые значения, программа пересчитает и отобразит новый цвет.
