Документация пользователя для 3D-приложения

Это приложение предназначено для создания, преобразования и визуализации 3D-объектов, а также для отображения их ортографических проекций на три основные плоскости: \(Oxy\), \(Oxz\), и \(Oyz\). Оно позволяет пользователю:
- Создавать 3D-объекты на основе вершин и рёбер.
- Применять преобразования к объектам: масштабирование, перемещение, поворот.
- Просматривать объекты в 3D-пространстве.
- Строить ортографические проекции объекта.

2. Основные функции приложения

1. Создание 3D-объекта:
   - 3D-объект задаётся через его вершины и рёбра.
   - Вершины — это массив точек в трёхмерном пространстве, определяющих положение объекта.
   - Рёбра — это наборы индексов, которые соединяют вершины, создавая структуру объекта.

2. Применение преобразований:
   - Масштабирование: позволяет изменять размеры объекта по осям \(X\), \(Y\) и \(Z\).
   - Перемещение: перемещает объект в пространстве, изменяя его положение по осям.
   - Поворот: позволяет вращать объект вокруг одной из осей (X, Y или Z) на заданный угол.

3. Визуализация 3D-объекта:
   - Объект отображается в 3D-пространстве с возможностью вращения и изменения масштаба.
   - Вершины объекта могут быть выделены для лучшего отображения.

4. Построение ортографических проекций:
   - Ортографическая проекция отображает объект без перспективного искажения.
   - Три основные проекции:
     - Проекция на плоскость Oxy: игнорируется координата Z.
     - Проекция на плоскость Oxz: игнорируется координата Y.
     - Проекция на плоскость Oyz: игнорируется координата X.

3. Шаги для использования приложения

1. Создание объекта:
   - Для создания объекта необходимо предоставить список вершин и рёбер.
   
2. Применение преобразований:
   - После создания объекта можно применить различные преобразования:
     - Масштабирование: Изменяет размеры объекта.
     - Перемещение: Смещает объект в пространстве.
     - Поворот: Вращает объект вокруг одной из осей.

3. Визуализация 3D-объекта:
   - После применения преобразований объект можно отобразить в 3D-пространстве с помощью библиотеки `matplotlib`.

   - Для просмотра ортографических проекций объекта можно использовать функцию `plot_orthographic_projections()`. Это создаст три графика, показывающих проекции объекта на плоскости:
     - \(Oxy\) (проекция на плоскость X-Y)
     - \(Oxz\) (проекция на плоскость X-Z)
     - \(Oyz\) (проекция на плоскость Y-Z)

4. Требования

Для работы приложения необходимо:
- Установленная библиотека `matplotlib` для визуализации.
- Установленная библиотека `numpy` для работы с массивами и матрицами.
