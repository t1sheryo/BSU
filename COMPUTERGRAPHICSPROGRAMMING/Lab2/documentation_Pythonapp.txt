Основные функции
Выбор директории: пользователь выбирает папку с изображениями, данные из которых будут обработаны и отображены в таблице.
Отображение метаданных: приложение извлекает и показывает такие параметры, как имя файла, размер изображения, разрешение (DPI), глубина цвета и сжатие.
Основные компоненты кода
Функция get_image_info(file_path)

Описание: открывает изображение и извлекает из него метаданные.
Параметры: file_path (str) — полный путь к изображению.
Возвращаемое значение: словарь с информацией об изображении (имя файла, размер, DPI, глубина цвета и сжатие).
Исключения: если файл не является изображением или происходит ошибка чтения, функция выводит сообщение об ошибке и возвращает None.
Функция scan_directory(directory)

Описание: рекурсивно сканирует указанную директорию, проверяя файлы на соответствие поддерживаемым форматам и вызывая get_image_info для каждого изображения.
Параметры: directory (str) — путь к папке.
Возвращаемое значение: список словарей с информацией об изображениях, которые были успешно прочитаны.
Функция open_directory()

Описание: открывает диалог выбора директории, сканирует выбранную папку и отображает метаданные в таблице.
Исключения: если пользователь не выбирает папку, функция не выполняет никаких действий.
Функция display_info_table(info_list)

Описание: очищает таблицу и добавляет строки с данными об изображениях.
Параметры: info_list (list) — список словарей с метаданными изображений.
Графический интерфейс
Основное окно:

Заголовок: "Информация об изображениях".
Размеры окна: 800x400 пикселей.
Элементы управления:
Кнопка "Выбрать папку с изображениями" — открывает диалог выбора директории.
Таблица для отображения метаданных — отображает информацию о каждом изображении в колонках: имя файла, размер (пиксели), разрешение (dpi), глубина цвета и сжатие.
Таблица (ttk.Treeview):

Колонки: "Имя файла", "Размер (пиксели)", "Разрешение (dpi)", "Глубина цвета", "Сжатие".
Настройка колонок: каждая колонка имеет заголовок и центрированное выравнивание для удобства восприятия.
Поддерживаемые форматы изображений
Приложение поддерживает следующие форматы: JPG, JPEG, GIF, TIF, BMP, PNG, PCX. Проверка форматов осуществляется по расширению файлов.

Пример использования
Запустите приложение, выполнив команду python image_info_gui.py.
Нажмите на кнопку "Выбрать папку с изображениями".
В появившемся диалоговом окне выберите папку с изображениями.
Приложение отобразит таблицу с информацией об изображениях в выбранной папке.
Возможные ошибки и их обработка
Если файл не является поддерживаемым изображением или невозможно прочитать метаданные, приложение выводит сообщение об ошибке в консоль.
Если пользователь не выберет директорию, приложение не будет выполнять никаких действий.
Дополнительные возможности для улучшения
Асинхронная обработка: использование многопоточности для ускорения обработки большого количества файлов.
Сортировка и фильтрация: добавление возможности сортировать и фильтровать данные в таблице.
Поддержка дополнительных форматов: добавление других форматов изображений (например, HEIC или WEBP).
