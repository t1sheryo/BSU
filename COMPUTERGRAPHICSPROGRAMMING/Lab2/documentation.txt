# Документация для приложения Image Metadata Reader

## Обзор

**Image Metadata Reader** — это настольное приложение, разработанное с использованием **Electron.js**. Приложение позволяет пользователям выбрать папку с изображениями и извлечь метаданные, такие как размеры, разрешение, глубина цвета и тип сжатия для поддерживаемых форматов изображений. Извлеченная информация отображается в таблице внутри интерфейса приложения.

## Основные возможности

- **Выбор папки с изображениями**: Пользователи могут выбрать целую папку с изображениями для одновременной обработки нескольких файлов.
- **Извлечение метаданных изображений**: Приложение извлекает ключевые метаданные из различных форматов изображений (JPEG, PNG, GIF, BMP, TIFF).
- **Отображение метаданных в таблице**: Метаданные для каждого изображения отображаются в динамически созданной таблице.
- **Поддержка нескольких форматов изображений**: Приложение поддерживает следующие форматы: JPEG, PNG, GIF, BMP и TIFF.
- **Разрешение (DPI) из EXIF-данных**: Для изображений с EXIF-метаданными (например, JPEG) извлекается разрешение (DPI), если оно доступно.
- **Динамическое обновление таблицы**: Таблица динамически обновляется при загрузке новых изображений и очищается перед загрузкой новых данных.

## Используемые технологии

- **Electron.js**: Основной фреймворк для разработки кроссплатформенных настольных приложений.
- **HTML/CSS/JavaScript**: Для построения интерфейса и отображения данных.
- **Node.js модули**: Используются для работы с изображениями и извлечения их метаданных.
  - `jpeg-js` для работы с JPEG изображениями.
  - `pngjs` для работы с PNG файлами.
  - `bmp-js` для обработки BMP изображений.
  - `utif` для чтения файлов TIFF.
  - `exif-js` для извлечения EXIF-данных.

## Поддерживаемые форматы изображений

1. **JPEG**: Извлекаются размеры изображения, разрешение (DPI) из EXIF-данных, глубина цвета (количество бит на компоненту) и тип сжатия.
2. **PNG**: Извлекаются размеры изображения, глубина цвета, тип сжатия (Deflate) и разрешение (если указано).
3. **GIF**: Извлекаются размеры изображения, глубина цвета (8-битная палитра) и тип сжатия (LZW).
4. **BMP**: Извлекаются размеры изображения, глубина цвета и отсутствие сжатия (BMP обычно не сжимается).
5. **TIFF**: Извлекаются размеры изображения, разрешение (если указано), глубина цвета и тип сжатия.

## Установка и запуск

1. Убедитесь, что на вашем компьютере установлены Node.js и npm.
2. Склонируйте или скачайте проект.
3. Выполните установку зависимостей с помощью команды:

   ```bash
   npm install
   ```

4. Для запуска приложения выполните команду:

   ```bash
   npm start
   ```

## Структура проекта

- **main.js**: Основной файл приложения, создающий окно и загружающий интерфейс.
- **src/index.html**: HTML-файл для интерфейса приложения.
- **src/style.css**: CSS-стили для оформления интерфейса.
- **src/app.js**: Логика обработки изображений и извлечения метаданных.

## Примечания

- Для работы с изображениями в разных форматах используются различные библиотеки Node.js.
- Разрешение (DPI) доступно только для форматов, которые поддерживают EXIF или другие метаданные, содержащие информацию о разрешении.

## Заключение

Приложение **Image Metadata Reader** упрощает процесс извлечения метаданных из изображений и предоставляет пользователям удобный способ просмотра этих данных для различных форматов изображений.
