# Lab 1 - React.js Компоненты

Этот проект содержит обновленные React компоненты для лабораторной работы 1.

## Обновления

### 1. Преобразование в функциональные компоненты
- **Example** - преобразован из классового компонента в функциональный с использованием React Hooks
- **Regions** - преобразован из классового компонента в функциональный с использованием React Hooks
- **TelephoneStation** - новый функциональный компонент для телефонной станции

### 2. Интеграция Material UI
Все компоненты теперь используют библиотеку Material UI для современного и красивого интерфейса.

### 3. Новый компонент TelephoneStation
Компонент для поиска сервисов телефонной станции по подстроке.

## Структура проекта

```
demo_react_lab1/
├── components/
│   ├── Example/           # Обновленный функциональный компонент
│   ├── Regions/           # Обновленный функциональный компонент
│   └── TelephoneStation/  # Новый компонент телефонной станции
├── modelData/
│   ├── example.js         # Модель данных для Example
│   ├── regions.js         # Модель данных для Regions
│   └── telephoneStation.js # Новая модель данных для телефонной станции
├── getting-started.html   # Страница для компонента Example
├── p2.html               # Страница для компонента Regions
├── p2_telephone.html     # Новая страница для компонента TelephoneStation
└── webpack.config.js     # Конфигурация с новыми точками входа
```

## Запуск проекта

### 1. Установка зависимостей
```bash
npm install
```

### 2. Сборка проекта
```bash
npm run build
```

### 3. Запуск веб-сервера
```bash
node webServer.js
```

### 4. Открытие страниц в браузере
- **Example компонент**: http://localhost:3000/getting-started.html
- **Regions компонент**: http://localhost:3000/p2.html
- **TelephoneStation компонент**: http://localhost:3000/p2_telephone.html

## Особенности компонентов

### Example
- Отображает ваше имя из модели данных
- Поле для изменения девиза
- Демонстрирует использование React Hooks

### Regions
- Поиск областей Беларуси по подстроке
- Автоматическое обновление результатов
- Использование Material UI List для отображения

### TelephoneStation
- Поиск сервисов телефонной станции по подстроке
- Красивый интерфейс с Material UI
- Отображение текущего поиска
- Сообщения об отсутствии результатов

## Технические детали

- **React 17.0.2** с функциональными компонентами и хуками
- **Material UI 5.15.0** для современного UI
- **Webpack 5** для сборки
- **Babel** для транспиляции JSX
